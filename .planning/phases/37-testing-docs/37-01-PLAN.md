### Task 1: E2E Test Suite — v11.0 Features

**File**: `tests/e2e/test_v11_e2e.py` (new, ~200 LOC)

Uses the existing `e2e_orchestrator` fixture from `test_orchestrator_e2e.py` (moved to conftest if needed).

```python
class TestAsyncRun:
    def test_async_run_returns_202_with_run_id(self, e2e_orchestrator):
        """POST /run → HTTP 202 with run_id + status=running."""

    def test_async_run_poll_completion(self, e2e_orchestrator):
        """POST /run → poll GET /runs?id=xxx every 0.5s, max 30s → completed."""

    def test_async_run_invalid_stages(self, e2e_orchestrator):
        """POST /run with stages=0 or stages=6 → HTTP 400."""

    def test_async_run_invalid_paper_id(self, e2e_orchestrator):
        """POST /run with paper_id="abc" → HTTP 400."""


class TestGetGeneratedCode:
    def test_get_generated_code_with_seeded_data(self, e2e_orchestrator):
        """Seed formulas + generated_code, GET /generated-code?paper_id=X returns entries."""

    def test_get_generated_code_language_filter(self, e2e_orchestrator):
        """GET /generated-code?paper_id=X&language=python filters correctly."""

    def test_get_generated_code_missing_paper_id(self, e2e_orchestrator):
        """GET /generated-code (no paper_id) → HTTP 400 VALIDATION_ERROR."""

    def test_get_generated_code_nonexistent_paper(self, e2e_orchestrator):
        """GET /generated-code?paper_id=99999 → empty list (not 404)."""


class TestGetRuns:
    def test_get_runs_list_empty(self, e2e_orchestrator):
        """GET /runs → empty list before any runs."""

    def test_get_runs_after_async_run(self, e2e_orchestrator):
        """POST /run → GET /runs → list contains run entry."""

    def test_get_runs_nonexistent_id(self, e2e_orchestrator):
        """GET /runs?id=run-nonexistent → HTTP 404."""
```

**Polling strategy**: `max_wait=30s`, `poll_interval=0.5s`, timeout raises `AssertionError("Run did not complete within 30s")`.

**Environment**: Uses `e2e_orchestrator` fixture (temp DB, real HTTP server, daemon thread, cleanup on teardown). Each test gets isolated DB via `tmp_path`.
