# Phase 46, Plan 02: Grafana dashboard

## Goal

Create Grafana dashboard with 6 panels for PePeRS pipeline observability.

## Prerequisites

- Plan 46-01 complete (Prometheus scraping PePeRS metrics)
- Grafana running at :3000 with VictoriaMetrics datasource

## Tasks

### Task 1: Create Grafana dashboard JSON

**File to create:** `/media/sam/1TB/monitoring-stack/grafana/dashboards/pepers-pipeline.json`

Provisioned dashboard (auto-loaded by Grafana, 30s refresh).

**6 Panels:**

1. **Service Health** (stat panel, row 0)
   - Query: `up{job="pepers"}`
   - Per-instance stat (green=1, red=0)
   - Grid layout: full width

2. **Papers Processed (24h)** (stat panel, row 0)
   - Query: `increase(pepers_papers_processed_total[24h])`
   - Single big number

3. **Latency per Stage** (bar gauge, row 1)
   - Query: `rate(pepers_pipeline_stage_duration_seconds_sum[5m]) / rate(pepers_pipeline_stage_duration_seconds_count[5m])`
   - Per stage label

4. **Error Rate** (time series, row 1)
   - Query: `rate(pepers_error_count_total[5m])`
   - By service + error_type

5. **Formula Validation** (stat + bar, row 2)
   - Query: `increase(pepers_formulas_validated_total[24h])` (stat)
   - Query: `increase(pepers_pipeline_stage_completed_total{stage="validator"}[24h])` by outcome (bar)

6. **Active Pipeline Runs** (gauge, row 2)
   - Query: `pepers_pipeline_runs_active`
   - Min 0, max 5, thresholds: green <1, yellow 1-3, red >3

**Dashboard metadata:**
- uid: "pepers-pipeline"
- title: "PePeRS Pipeline"
- tags: ["pepers", "pipeline", "research"]
- refresh: "30s"
- time range: last 24h

**Verification:**
- Dashboard loads at http://localhost:3000/d/pepers-pipeline
- All 6 panels render without errors (may show "No data" if pipeline hasn't run)
