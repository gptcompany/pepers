### Task 1: Strengthen formula filter (extractor/latex.py)

**Problem**: `MIN_FORMULA_LENGTH=3` + "has backslash or brace" is too permissive. Fragments like `\mu`, `\sigma`, `^{1}`, `\beta_t` pass through.

**Changes to `filter_formulas()`**:

1. Raise `MIN_FORMULA_LENGTH` from 3 to 10 characters
2. Add `is_trivial_formula()` helper that rejects:
   - Single LaTeX commands with no arguments beyond subscript/superscript: `\mu`, `\sigma`, `\alpha`, `\beta`, `\gamma`, `\delta`, `\epsilon`, `\theta`, `\lambda`, `\omega`, `\pi`, `\rho`, `\tau`, `\phi`, `\psi`, `\nu`, `\xi`, `\eta`, `\zeta`, `\kappa` (with optional `_t`, `_i`, `_{0}`, `^{2}` etc.)
   - Pure superscript/subscript fragments: `^{1}`, `_{i}`, `^{n+1}`
   - Single-token formulas that contain exactly one variable/command and zero operators
3. Add `has_operator()` check — formula must contain at least one of:
   - Arithmetic: `+`, `-`, `=`
   - LaTeX operators: `\frac`, `\sum`, `\int`, `\prod`, `\partial`, `\cdot`, `\times`, `\div`
   - Relations: `\leq`, `\geq`, `\neq`, `\approx`, `\equiv`, `\sim`
   - OR have at least 2 distinct free variables/commands (compound expression)

**Guard against false positives**: Named environment formulas (from Pass 1) get a relaxed filter — if the formula was inside `\begin{equation}...`, the author explicitly marked it as math, so we trust it even if short.

**Implementation**:
```python
MIN_FORMULA_LENGTH = 10

OPERATORS = {'+', '-', '='}
LATEX_OPERATORS = {
    r'\frac', r'\sum', r'\int', r'\prod', r'\partial',
    r'\cdot', r'\times', r'\div', r'\leq', r'\geq',
    r'\neq', r'\approx', r'\equiv', r'\sim',
}
GREEK_SINGLE = re.compile(
    r'^\\(alpha|beta|gamma|delta|epsilon|varepsilon|theta|lambda|'
    r'omega|pi|rho|sigma|tau|phi|psi|nu|xi|eta|zeta|kappa|mu|Gamma|'
    r'Delta|Theta|Lambda|Omega|Pi|Sigma|Phi|Psi)([_^]\{[^}]*\})*$'
)

def is_nontrivial(latex: str) -> bool:
    """Check if formula has enough complexity to be worth processing."""
    # Has arithmetic/relational operator
    if any(op in latex for op in OPERATORS):
        return True
    # Has LaTeX operator
    if any(op in latex for op in LATEX_OPERATORS):
        return True
    # Is just a single greek letter (possibly subscripted)
    if GREEK_SINGLE.match(latex.strip()):
        return False
    # Count distinct backslash commands (proxy for complexity)
    commands = set(re.findall(r'\\[a-zA-Z]+', latex))
    # Exclude formatting-only commands
    formatting = {r'\left', r'\right', r'\big', r'\Big', r'\bigg', r'\Bigg'}
    meaningful = commands - formatting
    return len(meaningful) >= 2
```

### Task 2: Add clean_latex() pre-processing (codegen/generators.py)

**Problem**: `parse_formula()` only strips `\begin/\end` and `$$/$`. Many LaTeX macros cause parse_latex() to fail or misinterpret.

**Add `clean_latex()` function** called at the start of `parse_formula()`:

```python
def clean_latex(latex: str) -> str:
    """Strip LaTeX macros unsupported by SymPy's parse_latex."""
    s = latex
    # 1. Remove \tag{...}, \label{...} (non-math annotations)
    s = re.sub(r'\\tag\{[^}]*\}', '', s)
    s = re.sub(r'\\label\{[^}]*\}', '', s)
    # 2. Unwrap text/formatting commands: \text{foo} → foo, \pmb{x} → x
    #    Handle one level of nesting: \text{for all} → for all
    for cmd in ('text', 'textit', 'textbf', 'textrm', 'mathrm', 'mathbf',
                'mathbb', 'mathcal', 'mathfrak', 'boldsymbol', 'pmb',
                'operatorname', 'bm'):
        s = re.sub(r'\\' + cmd + r'\{([^{}]*(?:\{[^{}]*\}[^{}]*)*)\}', r'\1', s)
    # 3. Remove spacing commands
    s = re.sub(r'\\[,;:!]', '', s)
    s = re.sub(r'\\(?:quad|qquad|hspace\{[^}]*\}|vspace\{[^}]*\})', '', s)
    # 4. Remove \left, \right (SymPy handles parens natively)
    s = re.sub(r'\\(?:left|right)[.|\[\](){}\\]?', '', s)
    # Alternative: \left. and \right| etc.
    s = s.replace(r'\left', '').replace(r'\right', '')
    # 5. Replace \equiv with = (semantically equivalent for codegen)
    s = s.replace(r'\equiv', '=')
    s = s.replace(r'\triangleq', '=')
    s = s.replace(r'\coloneqq', '=')
    s = s.replace(r'\defeq', '=')
    # 6. Remove \dots, \cdots, \ldots, \vdots, \ddots
    s = re.sub(r'\\[lcvd]?dots', '', s)
    # 7. Remove \displaystyle, \textstyle, \scriptstyle
    s = re.sub(r'\\(?:display|text|script|scriptscript)style', '', s)
    # 8. Remove \nonumber, \notag, \\, &
    s = s.replace(r'\nonumber', '').replace(r'\notag', '')
    s = re.sub(r'\\\\', ' ', s)  # line breaks → space
    s = s.replace('&', ' ')  # alignment markers → space
    # 9. Collapse whitespace
    s = re.sub(r'\s+', ' ', s).strip()
    return s
```

**Modify `parse_formula()`** to call `clean_latex()` after existing cleanup:
```python
def parse_formula(latex: str) -> sympy.Expr:
    # existing: strip \begin/\end, $$/$
    cleaned = re.sub(r'\\begin\{[^}]+\}|\\end\{[^}]+\}', '', latex).strip()
    cleaned = re.sub(r'^\$\$|^\$|\$\$$|\$$', '', cleaned).strip()
    cleaned = clean_latex(cleaned)  # NEW
    if not cleaned:
        raise ValueError("Empty LaTeX after cleanup")
    return parse_latex(cleaned)
```
