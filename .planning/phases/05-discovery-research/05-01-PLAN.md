# Plan 05-01: Discovery Service Research & Design

## Objective

Research arXiv, Semantic Scholar, and CrossRef APIs. Design Discovery service architecture, data flow, and integration patterns.

## Tasks

### Task 1: API Research
- Research arXiv API: query syntax, rate limits, response format, Python libraries
- Research Semantic Scholar API: endpoints, batch operations, fields, auth
- Research CrossRef API: DOI lookup, filtering, polite pool, rate limits
- Document findings in RESEARCH.md

### Task 2: Existing Codebase Analysis
- Analyze CAS microservice patterns from Phase 01
- Review shared library interfaces (BaseService, Paper model, DB layer)
- Identify reusable patterns and gaps

### Task 3: Design Decisions
- Decide on dependencies (arxiv pip package vs stdlib)
- Decide on HTTP client for S2/CrossRef (requests vs urllib)
- Design data flow: arXiv → S2 enrichment → CrossRef enrichment → DB
- Design error handling and rate limiting strategy
- Document in CONTEXT.md

## Acceptance Criteria

- [ ] RESEARCH.md with API details for all 3 services
- [ ] CONTEXT.md with all design decisions documented
- [ ] Dependencies chosen and justified
- [ ] Data flow designed
- [ ] Error handling strategy defined
- [ ] No open questions remaining
