## Implementation: `services/discovery/main.py`

### Functions

1. **`extract_arxiv_id(result)`** — Clean arxiv_id from `arxiv.Result`, strip version suffix (e.g. `2107.05580v2` → `2107.05580`)

2. **`search_arxiv(query, max_results)`** — Use `arxiv.Client(delay_seconds=3.0, num_retries=3)`, return list of paper dicts ready for DB insert. Filter out DataCite DOIs (prefix `10.48550`)

3. **`enrich_s2(arxiv_id)`** — GET `api.semanticscholar.org/graph/v1/paper/ARXIV:{id}` with fields: citationCount, referenceCount, influentialCitationCount, s2FieldsOfStudy, tldr, openAccessPdf, venue, publicationVenue, externalIds. Return enrichment dict or None. One retry on 429 with Retry-After header

4. **`enrich_crossref(doi)`** — GET `api.crossref.org/works/{doi}` with User-Agent `ResearchPipeline/1.0 (mailto:gptprojectmanager@gmail.com)`. Return message dict or None. Polite pool rate

5. **`upsert_paper(db_path, paper)`** — `INSERT INTO papers (...) VALUES (...) ON CONFLICT(arxiv_id) DO UPDATE SET title=excluded.title, abstract=excluded.abstract, ...`. Always return paper_id (new or existing). Uses `transaction()` from shared/db.py

6. **`update_paper_s2(db_path, paper_id, enrichment)`** — UPDATE paper with S2 fields (citation_count, reference_count, influential_citation_count, venue, fields_of_study, tldr, open_access_url, semantic_scholar_id)

7. **`update_paper_crossref(db_path, paper_id, crossref_data)`** — UPDATE paper.crossref_data with full CrossRef JSON blob

### Handler

`DiscoveryHandler(BaseHandler)` with single endpoint:

- **`POST /process`**: `{"query": "...", "max_results": 50}` → search arXiv, enrich each paper, return summary
- Validates: query (required string), max_results (1-500, default 50)
- Per-paper error isolation: individual failures logged and counted, don't block pipeline
- Rate limiting between enrichment calls: `time.sleep(1.0)` S2, `time.sleep(0.1)` CrossRef
- Returns:
  ```json
  {
    "papers_found": 3,
    "papers_new": 2,
    "papers_enriched_s2": 3,
    "papers_enriched_cr": 1,
    "errors": [],
    "time_ms": 12345
  }
  ```

### main()

config = load_config("discovery")  # port=8770, db_path
