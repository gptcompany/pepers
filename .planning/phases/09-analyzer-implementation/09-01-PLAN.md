## Implementation Steps

### Step 1: shared/models.py — Add REJECTED stage

Add `REJECTED = "rejected"` between ANALYZED and EXTRACTED in PipelineStage enum.

**Backward compatible:** Yes — new enum value, existing code doesn't reference it.

### Step 2: services/analyzer/__init__.py — Empty init

Create empty `__init__.py` for package.

### Step 3: services/analyzer/prompt.py — Scoring prompt

- `PROMPT_VERSION = "v1"` constant
- `SCORING_SYSTEM_PROMPT` — multi-line string with 5 criteria instructions + JSON schema
- `format_scoring_prompt(title, abstract, authors, categories)` — builds user prompt

Per DESIGN.md Section 4.

### Step 4: services/analyzer/llm.py — LLM client functions

- `_get_gemini_api_key()` — reads `GEMINI_API_KEY` from env
- `_strip_markdown_fences(text)` — regex strip ``` fences
- `call_gemini_cli(prompt, system, model, timeout)` — subprocess with stdin=DEVNULL
- `call_gemini_sdk(prompt, system, model, timeout)` — google-genai SDK
- `call_ollama(prompt, system, model, timeout, base_url)` — urllib POST to /api/generate
- `fallback_chain(prompt, system)` → `tuple[str, str]` — try in order, return (text, provider)

Per DESIGN.md Section 5. All functions use stdlib except SDK (google-genai).

### Step 5: services/analyzer/main.py — Handler + migration + main()

- `migrate_db(db_path)` — idempotent ADD COLUMN prompt_version
- `AnalyzerHandler(BaseHandler)` with class-level `threshold` and `max_papers_default`
- `@route("POST", "/process")` — full flow per DESIGN.md Section 6b:
  1. Parse request (paper_id, max_papers, force)
  2. Query papers from DB
  3. For each: build prompt → fallback_chain → parse JSON → validate scores → compute overall → update DB
  4. Return summary
- `main()` — load_config, init_db, migrate_db, set handler attrs, BaseService.run()

### Step 6: pyproject.toml — Add google-genai dependency

Add `"google-genai>=1.0"` to `[project] dependencies`.

## Risk Assessment

| Risk | Mitigation |
|------|-----------|
| google-genai import fails | Lazy import in call_gemini_sdk only |
| Gemini CLI not installed | Fallback chain handles gracefully |
| Ollama not running | Paper skipped, error logged |
| LLM returns invalid JSON | Retry once with strict suffix |
