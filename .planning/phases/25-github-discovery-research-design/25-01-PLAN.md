  <files>.planning/phases/25-github-discovery-research-design/DESIGN.md</files>
  <action>Document the multi-language GitHub search strategy:

  1. Query generation from paper context:
     - Primary: paper title (quoted) + `language:X`
     - Fallback: extracted keywords + `in:readme` + `language:X`
     - Quality filters: `stars:>N`, `pushed:>YYYY-MM-DD`, `archived:false`

  2. Multi-language search:
     - Run 3 parallel queries: language:python, language:rust, language:cpp
     - Merge and deduplicate by full_name
     - Sort by stars descending

  3. PAT authentication:
     - Header: `Authorization: Bearer $GITHUB_PAT`
     - Rate tracking: read x-ratelimit-remaining
     - Backoff: sleep until x-ratelimit-reset on 429

  4. Pagination: per_page=100, max 1000 results
