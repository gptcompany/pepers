## Tasks

### Task 1: Install antlr4-python3-runtime

**Goal**: Install `parse_latex()` dependency.

```bash
pip install "antlr4-python3-runtime>=4.11,<4.12"
python -c "from sympy.parsing.latex import parse_latex; print(parse_latex(r'\frac{1}{2}'))"
```

**Verification**: `parse_latex(r'\frac{1}{2}')` returns `1/2`.

### Task 2: Extract LLM client to `shared/llm.py`

**Goal**: Move 6 functions from `services/analyzer/llm.py` to `shared/llm.py` with enhancements.

**Changes from current analyzer/llm.py**:
1. `call_ollama()`: Add `format: str | dict = "json"` and `options: dict | None = None` params
2. `fallback_chain()`: Add `order: list[str] | None = None` param (default=None → Gemini-first)

**Files**:
- **Create**: `shared/llm.py` (~210 LOC) — all 6 functions with enhancements
- **Modify**: `services/analyzer/llm.py` — replace body with re-exports from shared/llm

**Verification**: Existing analyzer tests pass unchanged.

### Task 3: Add FormulaExplanation model to `shared/models.py`

**Goal**: Add Pydantic model for LLM explanation validation.

```python
class FormulaExplanation(BaseModel):
    explanation: str
    variables: list[dict[str, str]] = []
    assumptions: list[str] = []
    domain: str = ""
```

**Files**: Modify `shared/models.py`

### Task 4: Create `services/codegen/generators.py`

**Goal**: SymPy code generation module (~150 LOC).

**Functions**:
- `parse_formula(latex: str) -> sympy.Expr` — LaTeX→SymPy with environment stripping
- `generate_c99(expr, func_name) -> dict` — C99 via `codegen("C99")`
- `generate_rust(expr, func_name) -> dict` — Rust via `codegen("Rust")`
- `generate_python(expr, func_name) -> str` — Python via `pycode()`
- `generate_all(latex, formula_id) -> list[dict]` — orchestrator with per-language error isolation

**Files**: Create `services/codegen/generators.py`

### Task 5: Create `services/codegen/explain.py`

**Goal**: LLM explanation module (~100 LOC).

**Functions**:
- `explain_formula(latex, context, paper_title) -> dict | None`

**Uses**: `shared/llm.py` (call_ollama with structured output, fallback_chain)

**Files**: Create `services/codegen/explain.py`

### Task 6: Create `services/codegen/main.py`

**Goal**: CodegenHandler + DB operations + main() (~250 LOC).

**Pattern**: Copy validator/main.py pattern exactly.

**Endpoints**: POST /process (inherited /health, /status from BaseHandler)

**DB operations**:
- `_query_formulas()` — stage='validated', JOIN papers for title
- `_store_generated_code()` — DELETE+INSERT per formula+language
- `_update_formula_description()` — UPDATE formulas.description
- `_update_formula_stage()` — validated → codegen
- `_mark_formula_failed()` — set stage='failed'

**Files**:
- Create `services/codegen/__init__.py` (empty)
- Create `services/codegen/main.py`
