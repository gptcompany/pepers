### Task 2: Create `tests/unit/test_codegen.py`

**Goal**: Unit tests for generators.py + explain.py (~30 tests). All LLM calls mocked.

**Tests for `parse_formula()`**:
- Simple fraction: `\frac{1}{2}` → `1/2`
- Multi-variable: `\frac{p}{a} - \frac{q}{b}` → expr with free_symbols {a, b, p, q}
- Environment stripping: `\begin{equation}x^2\end{equation}` → `x**2`
- Dollar sign stripping: `$x^2$`, `$$x^2$$`
- Empty LaTeX → ValueError
- Invalid LaTeX → Exception

**Tests for `generate_c99()`**:
- Simple expr → has "double" in code, has header
- Variables list matches expr.free_symbols

**Tests for `generate_rust()`**:
- Simple expr → has "fn" in code

**Tests for `generate_python()`**:

**Tests for `generate_all()`**:
- Valid LaTeX → 3 results (c99, rust, python), all successful

**Tests for `explain_formula()`** (mocked LLM):
