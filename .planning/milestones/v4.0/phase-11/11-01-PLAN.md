### Task 2: RAGAnything Client Design

Design the HTTP client for RAGAnything service interaction.

**Deliverable:** Design spec for `services/extractor/rag_client.py` with:

```python
RAG_BASE_URL = "http://localhost:8767"
RAG_POLL_INTERVAL = 10  # seconds between polls
RAG_TIMEOUT = 600  # max wait for processing

def check_service(base_url: str = RAG_BASE_URL) -> dict
```
- GET `/status`
- Returns status dict with circuit_breaker state
- Raise `RuntimeError` if circuit breaker open or service unreachable

```python
def submit_pdf(pdf_path: Path, paper_id: str, base_url: str = RAG_BASE_URL) -> dict
```
- POST `/process` with `{"pdf_path": str(pdf_path), "paper_id": paper_id}`
- Handle 200 (cached) → return result directly
- Handle 202 (queued) → return `{"job_id": ..., "cached": False}`
- Raise on 400/404/429/503

```python
def poll_job(job_id: str, base_url: str = RAG_BASE_URL, timeout: float = RAG_TIMEOUT, interval: float = RAG_POLL_INTERVAL) -> dict
```
- GET `/jobs/{job_id}` every `interval` seconds
- Return completed result when `status == "completed"`
- Raise `TimeoutError` if `timeout` exceeded
- Raise `RuntimeError` if `status == "failed"`

```python
def read_markdown(output_dir: str) -> str
```
- Read markdown file from RAGAnything output directory
- Handle path mapping (`/workspace/1TB/` → `/media/sam/1TB/`)
- Return markdown text content

```python
def process_paper(pdf_path: Path, paper_id: str, base_url: str = RAG_BASE_URL) -> str
```
- Orchestrates: `check_service()` → `submit_pdf()` → `poll_job()` → `read_markdown()`
- Returns markdown text
- Single function for the handler to call

**Config env vars:**
| Env Var | Default | Purpose |
|---------|---------|---------|
| `RP_EXTRACTOR_RAG_URL` | `http://localhost:8767` | RAGAnything base URL |
| `RP_EXTRACTOR_RAG_TIMEOUT` | `600` | Max seconds to wait for processing |
| `RP_EXTRACTOR_RAG_POLL_INTERVAL` | `10` | Seconds between status polls |

### Task 3: LaTeX Extraction Engine Design

Design the regex-based LaTeX formula extraction module.

**Deliverable:** Design spec for `services/extractor/latex.py` with:

**Constants:**
```python
MATH_ENV_NAMES = ('equation', 'align', 'gather', 'multline', 'flalign', 'eqnarray', 'displaymath', 'math')
MIN_FORMULA_LENGTH = 3
CONTEXT_WINDOW = 200  # characters before/after formula
```

**Patterns (5 compiled regex, priority order):**
1. `PATTERN_NAMED_ENV` — `\begin{env}...\end{env}` with `re.DOTALL`
2. `PATTERN_DISPLAY_BRACKET` — `\[...\]` with `re.DOTALL`
3. `PATTERN_DISPLAY_DOLLAR` — `$$...$$` with `re.DOTALL`
4. `PATTERN_INLINE_PAREN` — `\(...\)` with `re.DOTALL`
5. `PATTERN_INLINE_DOLLAR` — `$...$` with lookbehind/lookahead guards

**Functions:**

```python
def extract_formulas(text: str) -> list[dict]
```
- 5-pass extraction with occupied-span tracking
- Returns list of `{"latex": str, "formula_type": str, "start": int, "end": int}`
- Sorted by document position

```python
def extract_context(text: str, start: int, end: int, window: int = CONTEXT_WINDOW) -> str
```
- Returns `text[max(0, start-window):end+window]`
- Strip to sentence boundaries if possible

```python
def filter_formulas(formulas: list[dict]) -> list[dict]
```
- Remove duplicates (by normalized LaTeX hash)
- Remove trivially short (`len(latex) < MIN_FORMULA_LENGTH`)
- Remove non-math (no `\` command = likely plain text or currency)

```python
def formulas_to_models(paper_id: int, text: str, raw_formulas: list[dict]) -> list[Formula]
```
- Convert extracted formulas to `Formula` Pydantic models
- Auto-computes `latex_hash` via model validator
- Sets `formula_type` = "inline" | "display"
- Sets `context` via `extract_context()`

### Task 4: Extractor Handler Design

Design the AnalyzerHandler class and /process endpoint.

**Deliverable:** Design spec for `services/extractor/main.py`:

```python
class ExtractorHandler(BaseHandler):
    @route("POST", "/process")
    def handle_process(self, data: dict) -> dict
```

**Process flow:**
1. Parse request: `paper_id` (optional, specific paper) or batch mode (all `stage='analyzed'`)
2. Optional: `max_papers` (default 10), `force` (reprocess)
3. Query DB: `SELECT * FROM papers WHERE stage='analyzed'` (or specific paper_id)
4. For each paper:
   a. Download PDF: `download_pdf(paper, pdf_dir, session)`
   b. Process with RAGAnything: `process_paper(pdf_path, paper.arxiv_id, rag_url)`
   c. Extract formulas: `extract_formulas(markdown)` → `filter_formulas()` → `formulas_to_models()`
   d. INSERT formulas into DB (deduplicate by `latex_hash` per paper)
   e. UPDATE `papers SET stage='extracted'`
   f. Rate limit: `time.sleep(download_delay)` between papers
5. Return summary:
```json
{
  "success": true,
  "service": "extractor",
  "papers_processed": 5,
  "formulas_extracted": 47,
  "papers_failed": 0,
  "errors": [],
  "time_ms": 35000
}
```

**Error handling per paper:**
- PDF download fails → `paper.stage = 'failed'`, `paper.error = "pdf_download: {err}"`, continue
- RAGAnything timeout → `paper.stage = 'failed'`, `paper.error = "raganything_timeout"`, continue
- RAGAnything circuit breaker open → abort batch, return 503
- 0 formulas found → `paper.stage = 'extracted'` (valid: paper has no math)
- DB insert error → `paper.stage = 'failed'`, `paper.error = "db_insert: {err}"`, continue

**Endpoints:**
| Endpoint | Method | Purpose |
|----------|--------|---------|
| `/health` | GET | Service status + RAGAnything status |
| `/status` | GET | Papers pending, formulas count, last processed |
| `/process` | POST | Process analyzed papers |
