# Plan 13-01: Extractor Service Test Suite

> Phase 13 deliverable. Comprehensive tests for the Extractor service (644 LOC across 4 modules).

## Goal

Create unit, integration, and E2E tests for the Extractor service following established patterns from Analyzer (v3.0).

## Test Matrix

| File | Type | What | Mock | Est. Tests |
|------|------|------|------|------------|
| `tests/unit/test_extractor.py` | Unit | pdf.py, rag_client.py, latex.py pure logic | requests.Session, urllib.request | ~35 |
| `tests/integration/test_extractor_db.py` | Integration | Full pipeline with real SQLite, mock HTTP | RAGAnything, arXiv HTTP | ~12 |
| `tests/e2e/test_extractor_e2e.py` | E2E | Real RAGAnything + sample arXiv PDF | None (real services) | ~3 |

**Target:** ~50 tests, ≥85% coverage on extractor modules.

## Tasks

### Task 1: Unit Tests — `tests/unit/test_extractor.py`

**1a. TestGetPdfUrl** (~4 tests)
- `test_uses_stored_pdf_url` — paper.pdf_url present → rewrite to export domain
- `test_rewrites_http_to_https_export` — http://arxiv.org → https://export.arxiv.org
- `test_rewrites_https_to_export` — https://arxiv.org → https://export.arxiv.org
- `test_fallback_to_constructed_url` — no pdf_url → construct from arxiv_id

**1b. TestDownloadPdf** (~5 tests, mock requests.Session)
- `test_cache_hit_skips_download` — existing file >1000 bytes → return immediately
- `test_successful_download` — mock 200 with PDF content-type
- `test_non_pdf_content_type_raises` — content-type not PDF → RuntimeError
- `test_creates_dest_dir` — dest_dir doesn't exist → mkdir
- `test_slash_in_arxiv_id_safe_name` — old-style arxiv_id (hep-ph/0601001) → underscore

**1c. TestCreateSession** (~2 tests)
- `test_session_has_user_agent` — verify headers
- `test_session_has_retry_strategy` — verify retry config

**1d. TestCheckService** (~3 tests, mock urllib.request.urlopen)
- `test_service_healthy` — normal response → return dict
- `test_circuit_breaker_open` — state="open" → RuntimeError
- `test_service_unreachable` — URLError → propagate

**1e. TestSubmitPdf** (~3 tests, mock urllib.request.urlopen)
- `test_cached_result` — response with cached=true
- `test_new_job_queued` — response with job_id (202)
- `test_request_format` — verify POST body contains pdf_path and paper_id

**1f. TestPollJob** (~4 tests, mock urllib.request.urlopen + time.sleep)
- `test_completed_immediately` — first poll returns completed
- `test_completed_after_retry` — pending → pending → completed
- `test_job_failed` — status=failed → RuntimeError
- `test_timeout` — never completes → TimeoutError

**1g. TestReadMarkdown** (~4 tests, real filesystem via tmp_path)
- `test_reads_largest_md_file` — multiple .md files → returns largest
- `test_container_path_mapping_1tb` — /workspace/1TB/ → /media/sam/1TB/
- `test_container_path_mapping_3tb` — /workspace/3TB-WDC/ → /media/sam/3TB-WDC/
- `test_no_markdown_raises` — empty dir → FileNotFoundError

**1h. TestProcessPaper** (~3 tests, mock all sub-functions)
- `test_cached_flow` — submit returns cached → read_markdown
- `test_new_job_flow` — submit → poll → read_markdown
- `test_no_output_dir_raises` — empty output_dir → RuntimeError

**1i. TestExtractFormulas** (~8 tests, pure function)
- `test_named_environment_equation` — \begin{equation}...\end{equation}
- `test_named_environment_align` — \begin{align*}...\end{align*}
- `test_display_bracket` — \[...\]
- `test_display_dollar` — $$...$$
- `test_inline_paren` — \(...\)
- `test_inline_dollar` — $...$
- `test_priority_no_overlap` — higher priority wins on overlapping spans
- `test_empty_text` — returns empty list

**1j. TestFilterFormulas** (~4 tests)
- `test_removes_trivially_short` — len < 3 → filtered out
- `test_removes_non_math` — no \ or { → filtered
- `test_deduplicates` — same latex twice → one result
- `test_passes_valid_formulas` — valid formulas pass through

**1k. TestExtractContext** (~2 tests)
- `test_context_window` — extracts ±200 chars around formula
- `test_context_at_boundaries` — formula at start/end of text

**1l. TestFormulasToModels** (~2 tests)
- `test_creates_formula_models` — returns list[Formula] with correct fields
- `test_latex_hash_computed` — Formula model auto-computes hash

### Task 2: Integration Tests — `tests/integration/test_extractor_db.py`
