# v6.0 Codegen Service

## Goal

LLM-powered plain-language explanation of validated formulas, Python code generation via SymPy, C99 code generation, and Rust code generation — completing the pipeline from math to production code.

## Phases

### Phase 17: Codegen Research & Design
- Research SymPy codegen API, Rust AST generation, LLM explanation prompts
- Design service architecture (3 modules: main, generators, explain)
- Design LLM client refactoring (shared/llm.py)
- Deliverable: DESIGN.md

### Phase 18: Codegen Implementation
- Create shared/llm.py (extract from analyzer)
- Implement generators.py (parse_latex → C99/Rust/Python codegen)
- Implement explain.py (LLM explanation with Ollama-first fallback)
- Implement main.py (CodegenHandler, DB ops, /process endpoint)
- Add FormulaExplanation model to shared/models.py

### Phase 19: Codegen Testing
- Unit tests for generators.py (parse, C99, Rust, Python)
- Unit tests for explain.py (prompt, fallback, validation)
- Integration tests for CodegenHandler (/process endpoint)
- E2E tests with real Ollama + SymPy
