## Overview

Add GitHub repository search and deep analysis using Gemini CLI (1M context) with dynamic prompts generated from paper context (title, abstract, formulas). Enables the /research skill to find existing implementations of academic papers on GitHub.

## Milestone Summary

**Key Decisions:**
- Gemini CLI for 1M context analysis (vs SDK which has lower limits)
- Dynamic prompt generation from paper context (title, abstract, formulas)
- Schema v2: github_repos + github_analyses tables
- SDK fallback with max_output_tokens for Gemini when CLI unavailable

**Issues Resolved:**
- Gemini CLI in Docker requires sdk fallback (no CLI in container)
- SDK max_output_tokens parameter needed for reliable responses
