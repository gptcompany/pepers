# Milestone v2.0: Discovery Service

**Status:** IN PROGRESS
**Phases:** 5-7
**Total Plans:** 2 (1 complete, 1 in progress, 1 pending)

## Overview

First microservice in the research pipeline. Searches arXiv for papers by configurable queries, enriches with Semantic Scholar metadata (citations, TLDR, fields of study), and optionally enriches with CrossRef (when journal DOI available). All data persists to SQLite via the shared lib from v1.0.
