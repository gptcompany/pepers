# Veo Prompt 02 — PePeRS Arcade + Hidden Rules

Create an 8-second vertical premium animated short, 9:16.

Absolutely no text.
No subtitles.
No voiceover.
No readable symbols.
No letters.
No numbers.
No logos appearing as flat static title cards.
No UI text.

If image reference is supported, use the provided PePeRS logo as the visual reference for the mascot identity.
The mascot must feel clearly inspired by the PePeRS brand:
frog-like, clever, mischievous, arcade-coded, but polished and cinematic.
Use the PePeRS visual palette:
dark purple / indigo base, gold accents, cyan-blue glow, rich contrast.

Goal:
Create a funny but smart short that makes mathematics feel like the hidden rule engine of reality.
This is not a kids cartoon.
It is a universal "math for everyone" short:
playful, elegant, surprising, and understandable to all ages.

Core gag:
The PePeRS mascot tries to make the world chaotic.
Mathematics calmly keeps restoring structure.
The joke is visual:
chaos keeps trying to win, but symmetry and invariant rules always reassert themselves beautifully.

Mathematics must be visible through images, not through text.
Use:
- mirror symmetry
- rotational symmetry
- geometric tilings
- graph paths
- deformation that preserves structure
- balanced vs unbalanced shapes
- elegant correction from disorder to order
No readable equations.

Timeline:
0.0s - 1.2s
A logo-inspired PePeRS frog mascot pops into frame with playful confident energy.
It is inside a dark glowing mathematical arcade world.
The world contains floating geometric objects, tiled patterns, clean lines, node graphs, and orbit arcs.

1.2s - 3.0s
The mascot mischievously pokes or kicks a glowing shape.
The shape becomes lopsided, crooked, asymmetrical, unstable.
Then another shape bends wrong.
Then a rotating polygon spins off-center.
This should feel funny through timing, not cartoon silliness.

3.0s - 5.3s
A hidden mathematical rule field activates:
mirror lines appear,
rotation centers lock in,
tiling snaps into place,
distorted shapes re-balance themselves.
The mascot looks surprised but amused.
This is the main visual joke:
math quietly fixes everything.

5.3s - 8.0s
The mascot tries one last tiny rebellious move.
Again the world self-corrects, but this time into a breathtaking final structure:
a luminous mathematical emblem where the PePeRS mascot is integrated into symmetry, geometry, and invariant motion.
The final feeling should be:
math is the secret engine, and it is welcoming, beautiful, and unstoppable.

Tone:
- witty, not silly
- funny, not childish
- premium, not cheap
- arcade energy, but refined
- mathematically meaningful, not random decoration

Visual language:
- strong silhouette clarity
- continuous shot preferred
- elegant timing
- premium glow
- subtle abstract Portuguese rhythm only through tiled geometry and blue-white pattern logic, not through cultural symbols or tourism visuals

Camera:
- mostly centered
- slight energy in first half
- subtle punch-in at final reveal
- no shaking
- no rapid cuts

Color palette:
- deep indigo / violet-black
- warm gold / amber
- bright cyan / blue accents
- soft pale highlights
- PePeRS-like brand energy

Audio:
- smart playful synth pulses
- tiny refined elastic hits on distortions
- satisfying harmonic correction sound when symmetry restores order
- uplifting final shimmer
- no meme sounds
- no childish boings

Negative prompt:
text, titles, subtitles, letters, numbers, readable equations, static logo card, generic frog mascot, childish comedy, cringe gamer culture, cheap arcade effects, visual clutter, blackboard, classroom, tourist Portugal imagery, flag colors, stock explainer style, generic AI surrealism, overcomplicated scenes, ugly low-detail animation
